# TodaysWod

## Description

Check out today's workouts and manage your reservations.

## Stack

1. React Native
2. Typescript
3. Style-Components
4. Redux

## Feature

1. Check workouts
2. Manage reservations
3. Check alert and membership
4. Login, out, sign up and manage workout center
